## errors="ignore" semantics — how ported (anchor #14)

`golden:180`: `path.read_text(encoding="utf-8", errors="ignore")` — invalid bytes dropped (not replaced with U+FFFD). `\n` (0x0A = valid UTF-8) preserved → line offsets intact.

Rust: `String::from_utf8_lossy(&raw).chars().filter(|&c| c != '\u{FFFD}').collect()`. This drops U+FFFD replacements that `from_utf8_lossy` would insert for invalid sequences, achieving exactly `errors="ignore"` semantics. Unit test (e): file with `\xff\xfe` prefix bytes + valid token on line 2 → token found at lineno=2.
